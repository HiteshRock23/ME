# ME — Product Principles

Every architectural and design decision must follow these principles.

---

## Capture

* Users never organize.
* Users never create folders.
* Users never create notebooks.
* Users never create categories.
* Users never create manual tags.
* Users never create titles.
* Capture should take less than 10 seconds.
* Every feature must reduce friction.

## Data Integrity

* Always save the raw memory first.
* Never lose user data.
* Never reject a memory because AI is unavailable.

## AI

* AI organizes everything.
* AI enhances memories.
* AI never replaces memories.

## Retrieval

* Retrieval should feel like talking to your own memory.
* Users search by meaning, not metadata.
* Everything is searchable.

## Architecture

* Design around the pipeline: Capture → Understand → Retrieve.
* Never design around CRUD.
* Every component must belong to one of the three pillars.
